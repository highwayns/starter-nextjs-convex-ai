# 目标成本与碳足迹分析

## Purpose

在设计阶段锁定成本与碳的70%+，做出权衡。

## When to Use

概念/设计冻结前/采购定点前。

## Process

### Step 1

建立目标成本分解（材料/加工/检验/运输/能耗）。

### Step 2

材料与工艺的碳强度评估与减排方案。

### Step 3

形成设计权衡与决策记录。

## Expected Outputs

- 目标成本分解表
- 碳足迹清单
- 权衡与决策记录
