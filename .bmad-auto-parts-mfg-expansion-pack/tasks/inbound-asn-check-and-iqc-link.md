# 收货与ASN校验/IQC联动

## Purpose

确保ASN/标签/实物一致并与IQC无缝衔接。

## When to Use

每批收货。

## Process

### Step 1

校验ASN字段、箱单/发票与条码。

### Step 2

Dock-to-Stock时钟启动，异常隔离与IQC叫检。

### Step 3

收货上架触发追溯链更新。

## Expected Outputs

- ASN校验记录
- Dock-to-Stock报表
- 异常与叫检记录
