# 规范与法规分解

## Purpose

把客户/法规/标准要求分解到图纸、公差、工艺与检验。

## When to Use

立项/新客户/法规更新。

## Process

### Step 1

建立需求映射矩阵：条款→图纸/工艺/检验。

### Step 2

识别特殊特性（CC/SC）并定义控制策略。

### Step 3

制定验证与证据清单（试验/报告/证书）。

## Expected Outputs

- 规范分解矩阵
- 特殊特性清单与控制策略
- 证据清单
