# PPAP 提交与PSW

## Purpose

按顾客要求输出PPAP包并获取PSW。

## When to Use

量产前/变更后。

## Process

### Step 1

准备DFM/DFA/DFMEA/PFMEA/控制计划/流程图/MSA/SPC/FAI等证据。

### Step 2

完成Run@Rate/SOR与能力指数。

### Step 3

提交并闭环问题直到PSW。

## Expected Outputs

- PPAP目录与状态
- SOR与能力报告
- PSW
