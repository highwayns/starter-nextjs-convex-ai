# 试验执行与偏差控制

## Purpose

规范试验执行、记录与偏差申报/评审/批准/恢复。

## When to Use

一切试验执行期间。

## Process

### Step 1

按SOP执行并实时记录（可追溯采集/照片/视频）。

### Step 2

偏差/中止/重测按流程审批并留痕。

### Step 3

样本与版本追溯闭环。

## Expected Outputs

- 试验记录包
- 偏差日志
- 样本追溯记录
