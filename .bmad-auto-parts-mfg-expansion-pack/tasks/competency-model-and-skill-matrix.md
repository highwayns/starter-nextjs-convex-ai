# 胜任模型与技能矩阵

## Purpose

建立岗位胜任模型与产线技能矩阵，支撑OJT与证书。

## When to Use

新岗位/产线变更/审计前。

## Process

### Step 1

定义岗位技能、行为与证照要求。

### Step 2

构建技能矩阵（A/B/C等级与双人覆盖）。

### Step 3

与生产/质量/EHS验证并发布。

## Expected Outputs

- 胜任模型文档
- 技能矩阵表
- 评审与发布记录
