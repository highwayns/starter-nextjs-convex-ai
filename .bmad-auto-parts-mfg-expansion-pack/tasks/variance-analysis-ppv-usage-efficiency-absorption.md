# 成本差异分析（PPV/用量/效率/吸收）

## Purpose

建立月度差异桥并定位根因，提出纠正行动。

## When to Use

月结后/异常波动。

## Process

### Step 1

拉取ERP/MES/采购价格与产出数据，生成差异桥。

### Step 2

分解材料PPV、用量、效率、制造费用吸收、废品等。

### Step 3

形成行动清单与责任/期限，跟踪闭环。

## Expected Outputs

- 差异桥报告
- 根因与行动清单
- 闭环与收益证明
