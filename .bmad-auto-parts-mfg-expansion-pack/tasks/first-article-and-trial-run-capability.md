# 首件与试生产能力评估

## Purpose

确保首件与小批试产满足尺寸/功能/节拍与质量能力。

## When to Use

NPI/ECN后复产。

## Process

### Step 1

组织FAI与过程首件，确认SOP/量具/治具/参数。

### Step 2

进行小批试产，评估FPY/Cp/Cpk与OEE。

### Step 3

形成缺口清单与对策。

## Expected Outputs

- FAI报告
- 试产能力报告
- 缺口与对策清单
