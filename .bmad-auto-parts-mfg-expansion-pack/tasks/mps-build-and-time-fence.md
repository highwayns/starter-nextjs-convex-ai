# MPS 主生产计划与时间围栏

## Purpose

基于需求/库存/能力构建MPS并应用时间围栏策略。

## When to Use

每周滚动与NPI/ECN切换前。

## Process

### Step 1

计算RCCP，识别瓶颈与负荷率。

### Step 2

按冻结/受控/自由区策略发布MPS。

### Step 3

与需求/供应变更保持版本与追溯。

## Expected Outputs

- MPS表
- 时间围栏与RCCP报告
- 版本与追溯记录
