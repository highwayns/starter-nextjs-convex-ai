# Milk-run 路线与补料

## Purpose

制定厂内Milk-run节拍与路线，实现准时补料。

## When to Use

物料拉动与节拍不稳定时。

## Process

### Step 1

规划线路/频次/车次与装载标准。

### Step 2

对接看板触发与超市补料规则。

### Step 3

试运行并优化停靠节拍与安全。

## Expected Outputs

- Milk-run路线图
- 节拍与装载标准
- 试运行复盘
