# SIOP 循环与统一口径

## Purpose

以一个数字计划（One Number Plan）对齐销售/运营/财务，形成月度SIOP节奏。

## When to Use

每月SIOP/重大波动。

## Process

### Step 1

准备需求/供给/库存/财务假设，识别差距。

### Step 2

召开SIOP会议并形成决议与责任。

### Step 3

回灌到MPS/MRP与采购/生产节奏。

## Expected Outputs

- SIOP议程与决议
- 假设与情景包
- 回灌清单
