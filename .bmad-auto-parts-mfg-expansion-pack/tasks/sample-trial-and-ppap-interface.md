# 样件/试制与PPAP接口

## Purpose

确保样件与PPAP要素按时达成并满足客户要求。

## When to Use

试制阶段/PPAP前。

## Process

### Step 1

跟踪样件BOM/图纸/工艺与时间点。

### Step 2

收集FAI/MSA/SPC/能力等PPAP证据。

### Step 3

闭环问题并准备PSW支持。

## Expected Outputs

- 样件计划
- PPAP接口跟踪表
- PSW支持证据
