# 逃逸分析与检测矩阵

## Purpose

识别缺陷逃逸路径并加固检测防线。

## When to Use

发生外部缺陷/客户抱怨后。

## Process

### Step 1

梳理检测/防错/抽样链路，识别薄弱点。

### Step 2

建立Detection Matrix并提升检出能力。

### Step 3

验证改善有效性并固化。

## Expected Outputs

- Detection Matrix
- 改善项与验证记录
