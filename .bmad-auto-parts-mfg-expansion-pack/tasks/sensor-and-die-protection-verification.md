# 传感器与模具保护验证

## Purpose

确保误送/件出/拉延/脱料/负载监控等传感器在线有效。

## When to Use

每天首件前/换模后/异常后。

## Process

### Step 1

逐项触发自检，验证报警与停机动作。

### Step 2

检查接线/插头/位置与信号稳定性。

### Step 3

确认参数阈值与旁路管理。

## Expected Outputs

- 传感器点检记录
- 报警与停机测试记录
- 旁路台账
