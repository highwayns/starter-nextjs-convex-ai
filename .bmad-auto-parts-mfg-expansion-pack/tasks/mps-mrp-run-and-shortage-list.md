# MPS/MRP 计算与短缺清单

## Purpose

运行主生产计划与物料需求计划，输出采购/生产建议与短缺。

## When to Use

每周/需求变更。

## Process

### Step 1

锁定时间栏，滚动MPS与计划参数。

### Step 2

运行MRP，生成PO建议/WO建议与异常。

### Step 3

形成短缺清单并驱动闭环。

## Expected Outputs

- MPS/MRP输出
- PO/WO建议
- 短缺清单与闭环记录
