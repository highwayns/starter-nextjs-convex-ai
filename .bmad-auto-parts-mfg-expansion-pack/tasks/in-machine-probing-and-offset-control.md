# 机内探针量测与偏置控制

## Purpose

通过探针对刀/工件定位/几何补偿，减少首件与漂移。

## When to Use

首件/换装/温漂/报警后复位。

## Process

### Step 1

制定探针流程与点位；权限/程序受控。

### Step 2

采集偏置与补偿，自动/人工阈值判定。

### Step 3

异常联动停机/复检/工程支援并留痕。

## Expected Outputs

- 探针计划与记录
- 偏置/补偿日志
- 异常处置记录
