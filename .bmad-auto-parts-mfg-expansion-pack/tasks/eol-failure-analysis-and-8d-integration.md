# EOL 失效分析与8D联动

## Purpose

对EOL不良进行快速定位与闭环。

## When to Use

EOL不良上升/客户反馈时。

## Process

### Step 1

分层诊断（设备/程序/物料/装配手法）。

### Step 2

建立DOE/复测与隔离策略。

### Step 3

与质量8D联动并回灌防错。

## Expected Outputs

- EOL失效分析报告
- 隔离与复测记录
- 8D联动回溯
